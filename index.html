Meus Projetos

FlixFlow

FlixFlow é uma aplicação de streaming em Flutter que permite pesquisar filmes e séries, adicionar favoritos e explorar categorias de conteúdos. Com integração Firebase, o projeto oferece uma experiência simples e intuitiva para centralizar e personalizar o entretenimento.

TradeShark

TradeShark.pt é uma plataforma de e-commerce para materiais de construção, desenvolvida com HTML, CSS, JavaScript e PHP. Oferece funcionalidades como carrinho de compras, gestão de utilizadores e modos de visualização, ideal para soluções de comércio online de construção.

FishFlow

FishFlow Dashboard: Plataforma para monitorar a qualidade de aquários domésticos. Com HTML, CSS, JavaScript, PHP e MySQL, fornece uma dashboard interativa para visualizar dados como pH, temperatura, turbidez e luz, com gráficos dinâmicos e verificação de conformidade.
